How to use the website

This is the main interface of the website, which shows the main map, and the search bar on top. You can type in any location as you would in google maps. This pulls from the google maps api, Therefore it is limited to countries that support google maps.

Begin by searching the location of the first point of travel, then it will look up 9 different places nearby that location. Within the 5 kilometer radius. The categories include: restaurant, museum, cafe, night club, clothing stores We are working on an added feature, refresh function. The data pulled is from the google maps places library as a JSON object containing the picture, the name, the address and the location. We are also planning to implement the other features that the API supports. Such as reviews, open and close hours to built a better user experience.

When the recomended boxes are hovered over, the name of the place and a "+" button is implemented to the desired location of route and be added to the list of selected locations. In the future we would like to implement the Google routes features to view routes on the maps. The images on the rights side recommended page mirrors on the the maps for easier user identification.

In order from left to right, is in the order of precedence. Include the first point of travel on the left most column and continue to the right. In the future, an added feature we look to add is a time feature. Including the time to travel and adjusting time as needed by the user.

When you hover over the location, the name of the place will pop-up. This will help user to identify easily the location they have selected before.
